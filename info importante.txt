MIGRACIONES

Para crear migraciones:
dotnet ef migrations add InitialCreate
dotnet ef database update

Tener en cuenta que cuando vayan a hacerse migraciones estar√≠a bueno separarlas

Para la base de datos Site: EDITAR EL NOMBRE DE LA MIGRATION
dotnet ef migrations add <migration_name> \
  --context SiteDbContext \
  --output-dir DAO/Migrations/Site

Para la base de datos Central: EDITAR EL NOMBRE DE LA MIGRATION
dotnet ef migrations add <migration_name> \
  --context CentralDbContext \
  --output-dir DAO/Migrations/Central


Para aplicar migraciones en la base de datos Central:
dotnet ef database update --context CentralDbContext
